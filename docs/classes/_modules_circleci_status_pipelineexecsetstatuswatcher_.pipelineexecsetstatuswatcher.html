Readonly finalStateNotifier

finalStateNotifier: Subject<PipeExecSetStatusNotification>

This RxJS Subject will emit an integer : => To notify the subscribers that all pipelines in this Pipeline Execution Set have reached a final execution state : Final State In the sense of Automata theory => The emited integer is the index of this Pipeline Execution State in the Execution Plan (Array)

The pattern with the RxJS Subject : A./ Subcribers subscribe to the Subject B./ this PipelineExecSetStatusWatcher will emit a PipeExecSetStatusNotification with this.finalStateNotifier.next() method, only after this.start() is invoked.

The ReactiveParallelExecutionSet subscribes to this RxSubject, so that it is notified when all Pipelines have reached a final execution state, with or without errors.

Private progressMatrix

progressMatrix: any []

This is the progress matrix for all pipeline executions in one {ParallelExecutionSet}, built by a src/modules/circleci/PipelineExecSetStatusWatcher.ts, not the whole execution plan, as understood by the CircleCiOrchestrator class. -- It is filled with all the HTTP JSON responses of Circle CI API HTTP request to trigger pipelines. -- Each entry in this array is of the following form :

{ pipeline_exec_number: '2', id: 'ef4264c2-f6f4-4cc4-a928-e7f89f3aff90', created_at: '2020-09-30T10:59:27.610Z', exec_state: 'pending' }

This method queries the CircleCI API and adds or updates each entry of the this.progressMatrix array's [workflows_exec_state] JSon property

each entry of the this.progressMatrix array matches a Circle CI Pipeline in a [ReactiveParallelExecutionSet] Parallel Execution Set

Now, when :

=> all Pipelines have reached a final execution state, => or any Pipeline has reach a final execution state with errors,

Then this method will : => build and log the execution report using {@link reporting.PipelineExecSetReport} => and call the next() method of the this.finalStateNotifier RxJS Subject, to notify its ReactiveParallelExecutionSet friend the Parallel Execution Set has reached an Execution Final State
